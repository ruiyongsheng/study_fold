## React + Antd 打造在线商城管理系统

使用方法：

```js
// 安装依赖
yarn add
或者
npm install

// 运行
yarn run dev
或者
npm run dev
```

3. 编写路由文件

5. 管理后台列表页面搭建
6. 管理后台表单页面搭建
7. 退出和登录页面开发
8. 登录判断

10. 网络请求封装
11. 列表数据接口对接

13. 数据修改功能实现

### 项目中使用到的其他插件

项目中使用到的插件列出如下

```bash
antd 			    #UI组件4.xx版本
react-router-dom     #路由
rudex			    #redux核心库
react-redux          #react中集成redux的插件
redux-thunk          #redux插件
axios                #网络请求
braft-editor         #富文本编辑器
```
